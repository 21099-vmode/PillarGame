## PillarGame — это динамичная многопользовательская игра-викторина в реальном времени, разработанная на движке Unity.

Игроки подключаются к общей сетевой сессии и оказываются на вершинах столбов, которые постепенно затапливает поднимающаяся снизу лава. Система транслирует вопросы, и задача участников — быстро и правильно отвечать на них, чтобы удерживать свои позиции. Ошибка или медлительность приводят к опусканию столба. Выбывшие игроки телепортируются в изолированную зону наблюдения, а сессия автоматически завершается, когда на арене не остается живых участников, после чего выводится итоговый топ-лидеров.


## Управление

 `W` `A` `S` `D` — Перемещение персонажа (наземный бег во время раунда / свободный полет в режиме наблюдения).
 `Мышь`** — Управление обзором камеры (оглядеться по сторонам).
 `Левая кнопка мыши (ЛКМ)` — Взаимодействие с игровым интерфейсом (UI), выбор вариантов ответов на вопросы викторины, навигация по кнопкам меню.

## Как запустить
Вариант 1: Запуск готового билда (Для игроков / Проверяющего)

    Перейдите в раздел Releases на панели справа в этом репозитории и скачайте архив PillarGame_v1.0_Build.zip.

    Распакуйте архив в любую удобную папку на компьютере.

    Убедитесь, что вы и ваши друзья подключены к одной сети.

    Запустите файл PillarsGame.exe.

    Один из участников выбирает роль Host (Сервер + Игрок), остальные вводят его IP-адрес и нажимают Connect to Lobby (Подключиться).

Вариант 2: Запуск из исходного кода (Для разработчиков)

    Склонируйте данный репозиторий: git clone https://github.com/21099-vmode/PillarGame.git

    Откройте папку проекта через менеджер Unity Hub.

    В папке Assets/Scenes откройте сцену MainMenu.unity и запустите режим игры (кнопка Play).
